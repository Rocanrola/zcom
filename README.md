![](img/zcom.png)

Create, develop and distribute React components.

## Setup
```bash
npm install -g zcom
```

## Create a component
```bash
zcom my-component
```

## Run the dev environment (Storybook)
```bash
zcom dev
```
## Patch and publish (NPM)
```bash
zcom pap
```

## Just build
```bash
zcom build
```

## Build the storybook for distribution
```bash
zcom build-storybook
```
